# Vivarium — AI Orientation

**Vivarium** is a live C# scripting environment running as an
[MCP](https://modelcontextprotocol.io) server. It gives you a persistent,
stateful workspace where code and data accumulate across interactions.

## Which guide do you need?

- **[README_AI_USAGE.md](README_AI_USAGE.md)** — How to use Vivarium
  effectively: mental model, workflow patterns, tool reference, tips for
  minimizing context-window cost.

- **[README_AI_CONTRIBUTING.md](README_AI_CONTRIBUTING.md)** — How the code
  works internally: source map, architecture decisions, how to add tools,
  change metadata, or extend the system.

For the human-facing overview, see [README.md](README.md).
